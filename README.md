# Krzywe_Beziera
Program pobiera punkty kontrolne z pliku Dane.txt, na ich podstawie oblicza punkty na krzywej i wykorzystuje biblioteke matplotlib by wyświetlić krzywe Beziera.
Dane.txt zawierają punkty kontrolne krzywych które tworzą inicjały K N.<br/>
<img src="/img/inincjaly.png" width="350" alt=inicjaly><br/>
<h3>Struktura pliku Dane.txt</h3>
Plik Dant.txt zawiera punkty kontrolne.
W każdej lini są punkty do jednej krzywej Beziera przedzielone przecinkiem.
Linia zawiera 8 liczb czyli 4 punkty o kordynatach x i y. Nieparzyste to kordynaty x, parzyste y. Dwie pierwsze liczby to początek krzywej, dwie ostatnie to koniec.

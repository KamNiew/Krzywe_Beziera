# Krzywe_Beziera
Program pobiera punkty kontrolne z pliku Dane.txt, na ich podstawie oblicza punkty na krzywej i wykorzystuje biblioteke matplotlib by wyświetlić krzywe Beziera.
Dane.txt zawierają punkty kontrolne krzywych które tworzą inicjały K N.
